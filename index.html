Desktop PDF Converter

Wizard konversi PDF ke EPUB untuk Kindle.

Sekarang flow-nya bisa jalan untuk single file atau batch queue, dengan cover otomatis, metadata lengkap, mode OCR, dan target device Kindle yang lebih spesifik.
